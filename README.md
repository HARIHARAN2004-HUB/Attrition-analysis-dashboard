# Attrition-analysis-dashboard
1. Project Title:
Employee Attrition Analysis using Excel & Power BI

2. Short Description:
Developed an Employee Attrition Analysis dashboard to identify workforce turnover trends and understand factors influencing employee attrition. The project provides actionable HR insights through interactive visualizations and data-driven analysis using Excel and Power BI.

3. Tech Stack:
-Microsoft Excel
-Power BI
-DAX (Data Analysis Expressions)
-Data Visualization
-Business Intelligence (BI)

4. Data Source:
Custom HR Employee Dataset (1000 Employee Records)
Dataset includes employee demographics, job information, salary details, business travel, education, and attrition status.
Dataset Attributes
[Employee ID
Age
Gender
Department
Job Role
Education
Monthly Salary
Business Travel
Distance From Home
Attrition Status]

5. Features / Business Questions Solved:
**Employee Overview**
What is the total number of employees?
What is the overall attrition rate?

**Attrition Analysis**
Which departments have the highest attrition?
Which job roles experience the most employee turnover?
How does attrition vary across different age groups?

**Demographic Analysis**
Is attrition higher among male or female employees?
Which education fields show higher attrition rates?

**Salary Analysis**
Does salary influence employee attrition?
Which salary bands have the highest turnover?

Dashboard Features
-Interactive filters and slicers
-Department-wise analysis
-Gender-wise analysis
-Salary band comparison
-Attrition KPI tracking
-Dynamic visualizations for HR decision-making
